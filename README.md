# Midori-E-Commerce-App
Midori is One Stop Shop For All Your Garden Needs.

## Features
- Home Page
- Product Listing Page
- Filters by
  - Price
  - Category
  - Ratings
- Cart Management
- Wishlist Management
- Authentication
  - Sign-up
  - Login
  - Logout
 
#### App is not responsive as of now.

## Tech Stack
1. HTML
2. CSS
3. JS
